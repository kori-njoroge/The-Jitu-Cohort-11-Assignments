Q3

Outline the various stages of the React.js lifecycle.

(1.) Mounting

This method occurs when component is first added to the Dom.
Method called:
useEffect(): This hook is used to perform setup logic when the component is first mounted, such as fetching data or adding event listeners.

(2.) Updating

This phase occurs when the component receives new props or its state is updated.
Method called:
useEffect(): In this stage useEffect hook can also be used to perform update logic, such as re-fetching data or updating event listeners. This hook also excecutes the code inside when the dependecy array changes.

(3.) Unmounting

This stage occurs when a component is removed from the DOM.
Method called:
useEffect(): In this stage the useEffect hook is used to perform cleanup logic, such as removing event listeners,changing the state of a dependency array, when the component is unmounted useEffect performs the clean up logic if a clean up function is provided.
