Q1

What is the difference between state and props?

State
state is a way to store and manage local state within a component(the status of a component at a given time). State can be modified by the component, and changes to state trigger re-renders of the component. The state is declared and initialized within the component's class constructor or using the useState hook in case of functional components.

functional components
const[count, setCount] = useState(0);
The "setCount" is the method that update the state called count. State can also be passed as a prop.



Props
props are short form of properties, they are a way to pass data from a parent component to a child component. They are read-only, meaning that the child component cannot modify the props it receives. Props are passed down to a component using the component's tag.
i.e; < Count value ={count} /> The Count is an imported component to a parent component.

Receiving the passed props
export default function Count(props){ console.log(props.value).// log the value which is the count. }

Q4

Difference between controlled and uncontrolled components

In React, a "controlled component" is a form element that is controlled by React state, while an "uncontrolled component" is a form element that is controlled by its own internal state.

Controlled Component:
In a controlled component, the value of the form element is set by React state, and any updates to the input are reflected in the state. This means that the state is the source the form data, and the form elements are updated whenever the state changes.
To create a controlled component, you would use onChange handlers to update the state whenever the form element's value changes, and set the value property of the form element to the value stored in the state.

Uncontrolled components
In an uncontrolled component, the form element manages its own internal state and the React state is not updated when the form element changes.
To create an uncontrolled component, you would omit the value and onChange properties on the form element.

Explain what three dots mean in react

Known as the "spread operator" is the spread syntax in JavaScript.
The spread operator allows you to spread an array or an object into another array or object. In React, it's often used to pass down properties from a parent component to a child component. Also used with arrays to spread the elements of one array into another.
i.e;
let Arr1 = [1,2,3,4];
let arr2 =[5,6,7,8];
let arr3 =[...arr1, ...arr2]// the array 3 contains the elemnts from arr1 and arr2 respectively;
